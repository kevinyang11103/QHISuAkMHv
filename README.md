# 海滨体育馆管理系统 java1044

## 项目概述

海滨体育馆管理系统是一款基于前后端分离架构的信息管理系统。它主要包括管理员与学生两个角色，涵盖个人中心管理、器材管理、借还管理、校队签到、活动预约等多个功能模块，旨在提高体育馆的管理效率和服务水平。

## 技术栈

- 后端：Spring Boot
- 前端：Vue 2 + ElementUI
- Node.js版本：14

## 运行环境

- JDK版本：1.8
- 数据库版本：MySQL 5.7及以上
- 开发工具：IDEA、VSCode

- 个人中心管理
  - 账号信息管理
  - 密码修改
  - 个人信息查看
- 学生管理
  - 账户信息增删改查
  - 注册审核
  - 信息更新
- 器材管理
  - 器材录入
  - 信息更新与维护
- 器材借出管理
  - 借用申请处理
  - 审批与记录
- 器材归还管理
  - 归还流程管理
  - 库存信息更新
- 器材分类管理
  - 器材分类与标签
- 校队签到管理
  - 训练签到
  - 考勤记录
- 进入登记管理
  - 访客或使用者入场登记
- 离开登记管理
  - 使用者离场登记
- 活动预约管理
  - 活动场地预订
  - 时间表安排
- 灯光保修管理
  - 维修请求处理
- 体育论坛管理
  - 内容监管
  - 帖子与评论管理
- 系统管理
  - 用户权限管理
  - 数据备份

## 部署步骤

1. 环境搭建：安装JDK 1.8、MySQL 5.7以上版本、Node.js 14。
2. 后端配置：使用IDEA配置Spring Boot项目，连接MySQL数据库。
3. 前端配置：使用VSCode配置Vue项目，安装相关依赖。
4. 项目构建：编译后端代码，打包前端资源。
5. 部署应用：将打包后的代码部署到服务器。
